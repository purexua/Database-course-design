# hello_demo

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
## Project dependencies

- npm i vite-plugin-vue-setup-extend -D
> setup 语法糖

- npm i vue-router
> 添加路由

- npm i pinia
> 状态管理

- npm i mitt
> 组件间通信 类 pubsub

- npm i nanoid 
> 生成 唯一 id 工具
